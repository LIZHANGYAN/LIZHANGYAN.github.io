基础

注释

注释是任何存在于#号右侧的文字，主要作用是写给读者看的笔记

字面常量

一个字面常量（Literal Constants）的例子是诸如5、1.23这样的数字，或者是如这是一串文本 或 This is a string 这样的文本。

数字

  • 整数 Integers
  • 浮点数 Floats

字符串

一串字符串（String）是字符（Characters）的序列（Sequence）。
字符串是不可变的。

引号

  • 单引号

    'Quote me on this'

  • 双引号

    "What's your name?"

  • 三引号
    用来指定多行字符串，可以在三引号之间自由换行。

    格式化方法

    使用format()方法从其他变量构建字符串。
    如，str_format.py:

转义序列

  • 换行符\n
  • 制表符\t
  • 单引号\'
    What\'s your name?
  • 原始字符串
    未经特殊处理的字符串，比如转义序列，要在字符串前加r或R, 例如
    r"Newlines are indicated by \n"

    使用正则表达式时应全程使用原始字符串

变量 Veriables

变量的值可以变化

对象 Object

Python是强面向对象的，因为所有的一切都是对象，包括数字、字符串与函数.

逻辑行与物理行

  • Physical Line

    编写程序是所看到的内容

  • Logical Line

    Python编辑器所看到的单个语句

  • 显式行连接
    如果有一行非常长的代码，可以用反斜杠将其拆分成多个物理行

  • 隐式行连接
    逻辑行以括号开始。

缩进

空白区在Python中十分重要。逻辑行的开头留下空白区（空格或者制表符）用以确定行的缩进级别，并且确定语句分组。

具有相同缩进的语句被称为块(block)

运算符与表达式

运算符

  • +
  • -
  • *
  • ** 乘方
  • /
  • // 整除

    x 除以 y 并对结果向下取整

  • %
  • << 左移

    二进制

  • >> 右移

    二进制

  • & 按位与
  • | 按位或
  • ^ 按位异或
  • ~ 按位取反
  • <, <=
  • >, >=
  • ==, !=
  • not, and, or

    数值运算与赋值的快捷方式

  • 变量 = 变量 运算 表达式 –> 变量 运算 = 表达式

  • 求值顺序

    最好使用圆括号进行分组

    • ambda: Lambda 表达式
    • if - else: 条件表达式
    • or
    • and
    • not x
    • in, not in, is, is not, <, <=, >,>=, !=,==
    • |

      上述优先级从下往上越来越高

控制流

Python中有三种控制流语句-if,for 和 while。

if 语句

if 语句用以检查条件：如果条件为真（True）

while 语句

while 语句当条件为真时重复执行某块语句

for 语句

for…in 语句会在一系列对象上进行迭代（Iterates）。

range() 函数生成数字序列

函数

函数 是指可重复使用的程序片段。通过关键词def来定义。

  • 案例一 无参函数

  • 案例二 带参函数

    只在声明的函数内起作用

  • global 语句
    通过关键字global，确定变量是在最外面的代码块中定义的
